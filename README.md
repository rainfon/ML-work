# 基于三支决策的主动学习方法在多准则有序分类问题中的应用

本仓库为《高级机器学习理论》课程报告对应的实验代码和数据集。代码实现了面向多准则有序分类问题的偏好学习与主动学习实验，主要比较基础分类模型、Lex-PL 偏好学习模型以及基于三支决策的主动学习方法 TWD-AL。

## 1. 项目内容

本项目主要包括两个实验：

1. 静态分类性能比较
   比较 CART、KNN、SVM、Random Forest 和 Lex-PL 在九个多准则有序分类数据集上的分类性能。

2. 主动学习策略比较
   比较 AL-Rand、AL-Entropy 和 TWD-AL 三种主动学习策略在有限反馈预算下的分类效果。

其中，TWD-AL 以 Lex-PL 为基础学习器，通过模型支持度估计候选样本推荐类别的稳定性，并利用三支决策思想将候选样本划分为正域、边界域和负域，优先选择高风险样本进行主动询问。

## 8. 方法简介

Lex-PL 是一种基于加性效用函数和类别阈值的偏好学习模型。该模型首先通过最小化参考样本的赋类违反程度学习可行偏好模型，再在不增加违反程度的前提下最大化类别间隔，从而提升模型的稳定性和可解释性。

TWD-AL 在 Lex-PL 的基础上进一步引入模型支持度和三支决策机制。对于每个候选样本，程序通过模型采样估计其当前推荐类别的支持度，并依据三支决策风险阈值将样本划分为正域、边界域和负域。其中，负域样本表示当前推荐类别缺乏充分支持，因此被优先提交主动询问。获得反馈后，程序将被询问样本加入训练集，并重新训练分类模型。

## 9. 评价指标

实验采用以下评价指标：

```text
Accuracy：分类准确率，越大越好
Precision：精确率，越大越好
Recall：召回率，越大越好
Fmeasure：F1 值，越大越好
MAE：平均绝对误差，越小越好
```

由于多准则有序分类中的类别具有顺序关系，MAE 用于衡量预测类别与真实类别之间的顺序偏离程度。
